COMO ESTARÃO AS LINGUAGENS DE PROGRAMAÇÃO NO FUTURO?

As linguagens de programação evoluem cada vez mais, a fim de que facilitem o uso delas no dia a dia. Por isso, é provável que as linguagens voltadas para a web continuem se modificando para acompanhar as tendências que a cada dia surgem e se tornarem cada vez mais eficientes, seguras e acessíveis. Com o avanço da internet mais pessoas terão acesso a essas linguagens e mais usadas elas serão. Surgirá também novas linguagens adaptadas para as tecnologias que tiver quando elas surgirem. Com a evoluçao das linguagens os frameworks irão evoluir junto para também facilitar a programação como um todo. Ainda assim é possível que as linguagens basicas continuem sendo fundamentais e usadas, talvez não mais com a mesma frequência de agora, mas ainda usadas.
